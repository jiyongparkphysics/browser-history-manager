# v0.1.1 Release Draft

## Summary

First public release of Browser History Manager.

This release includes the current CLI and desktop GUI, project documentation, tests, and release assets.

## Highlights

- Search, filter, preview, export, and batch-delete browser history.
- Windows GUI and CLI builds are included.
- App icon and branding assets are included.
- Regression tests cover GUI pagination, selection, and backup management behavior.

## Verification Notes

- This release was prepared and runtime-verified on Windows.
- Current named browser targets in code are Brave, Chrome, Edge, Chromium on Linux, and Opera on Windows.
- Other Chrome-family browsers may work when a compatible History database is provided directly via `--db`, but they are not runtime-verified in this release.
- macOS and Linux CLI binaries are included as convenience builds and were not runtime-verified in this release.
- If WebView2 is missing, the Windows GUI build can prompt to install it on first launch.
